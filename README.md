# 💇🏽‍♀️ Karrie Beauty Hub - Serverless Booking System

A real-world serverless booking system built for a salon business using AWS and DevOps practices.

---

## 🚀 Features

- Customers can book appointments online  
- Bookings are stored in DynamoDB  
- Admin dashboard to view bookings  
- Delete bookings functionality  
- Fully deployed on AWS  

---

Admin Dashboard:

---

## ☁️ AWS Services Used

- API Gateway (HTTP API)  
- AWS Lambda  
- DynamoDB  
- S3 (Static Website Hosting)  

---

## ⚙️ DevOps

- CI/CD pipeline using GitHub Actions  
- Automatic deployment to S3 on every push
